## FIX: COMMENT INDICATOR — ORANGE DOT INSTEAD OF TRIANGLE ON READINESS COLUMN

The comment indicator is rendering as an orange circle/dot on the readiness column instead of a triangle. All other columns correctly show the triangle. Fix the readiness column cell to also render the triangle correctly.

### THE PROBLEM

- Most columns: comment indicator shows as ▲ triangle ✅
- Readiness column: comment indicator shows as ● orange dot ❌

The readiness column cell likely has CSS that is overriding or conflicting with the triangle's CSS. Common causes:

1. The readiness cell has `overflow: hidden` clipping the triangle
2. The readiness cell has `border-radius: 50%` turning the triangle into a circle
3. The readiness cell has different positioning that breaks the triangle shape
4. The readiness cell is using a different indicator element (like a `<span>` with `border-radius: 50%`) instead of the triangle CSS

Key properties that make it a triangle (NOT a circle):
- `width: 0` and `height: 0` — element has no dimensions
- `background: transparent` — no background fill
- `border-left: 8px solid transparent` — invisible left border creates the angle
- `border-top: 8px solid #fd5108` — visible top border creates the triangle shape
- `border-radius: 0` — NO rounding (this is likely the bug)
