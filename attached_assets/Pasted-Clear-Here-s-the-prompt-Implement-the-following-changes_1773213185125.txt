### LEFT COLUMN — Order Identifiers

Line 1: **FPR: [value]**
- "FPR:" label in regular grey text
- Value in bold, dark text
- Same size as current FPR display

Line 2: **FG: [value]**
- "FG:" label in regular grey text
- Value in smaller, lighter grey text (existing hierarchy)

Line 3: **SFG: [value]**
- "SFG:" label in regular grey text
- Value in smaller, lighter grey text (existing hierarchy)

---

### MIDDLE COLUMN — Production & Scheduling Details

Line 1: **Production time**
- Display the production hours/minutes for this individual order
- Format: "25 min", "1 hr 3 min", "4 hrs 18 min"
- Regular text, dark grey

Line 2: **CD: [Completion Date]**
- "CD:" label in grey text
- Completion Date value in dark text
- Format: "CD: March 19, 2026"
- If no Completion Date (not yet calculated): "CD: —" in grey

Line 3: **CT: [Completion Time]**
- "CT:" label in grey text
- Completion Time value in dark text
- Format: "CT: 01:43 AM" (12-hour format with AM/PM)
- If no Completion Time (not yet calculated): "CT: —" in grey

Line 4: **Availability: [value]**
- "Availability:" label in grey text
- Value depends on the order type:
  - Valid date: "Availability: March 19, 2026" — dark text
  - Non-date: "Availability: prio replenish" — orange text (#fd5108)
  - Non-date: "Availability: safety stocks" — orange text (#fd5108)
  - Non-date: "Availability: for sched" — orange text (#fd5108)
- The Availability label is always grey; only the VALUE changes color based on type.

### RIGHT COLUMN — Volume + Remove Button

**Volume display (right-aligned):**

- Same font size as the FPR value (matching size as requested)
- Bold, dark text
- Right-aligned
- Example: "8", "12", "20", "80"

Line 2: **metric tons (MT)** — Unit label
- Smaller font size
- Grey text (#a1a8b3)
- Right-aligned
- Appears on EVERY order row

**Remove button (×):**
- Small × button positioned to the right of the volume number (Line 1)
- Same behavior as current: click to exclude order from combine group
- Grey by default, red on hover
