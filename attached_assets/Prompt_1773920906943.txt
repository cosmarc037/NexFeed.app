**Validation:**
- The user must type exactly "Confirm" (case-sensitive, capital C).
- The [Delete] button is DISABLED until the input matches "Confirm" exactly.
- As the user types, check in real-time:
  - Input is empty or doesn't match → [Delete] button remains disabled (greyed out).
  - Input matches "Confirm" exactly → [Delete] button becomes enabled (turns red).

**Visual feedback while typing:**
- Partial match (e.g., "Con") → input border stays grey, no error, button disabled.
- Wrong text (e.g., "confirm" lowercase, "delete", etc.) → subtle red border on input (#fecaca), small error text below: "Please type 'Confirm' exactly." in red (#e53935), 10px.
- Exact match "Confirm" → input border turns green (#43a047), [Delete] button enables.
