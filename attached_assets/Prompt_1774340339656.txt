## 6. LINE — FIXED RECOMMENDATION BASED ON HIGHEST RUN RATE

### DROPDOWN — FILTERED BY FEEDMILL

| Feedmill | Lines shown in dropdown |
|---|---|
| Feedmill 1 | Line 1, Line 2 |
| Feedmill 2 | Line 3, Line 4 |
| Feedmill 3 | Line 5, Line 6 |
| Powermix | Line 7 |

### RECOMMENDATION LOGIC — ALWAYS FIXED

The recommendation ALWAYS shows the objectively best line based on Knowledge Base run rates. It does NOT change based on the user's current selection. Higher run rate = best.

```javascript
function getLineRecommendation(product, feedmillLines) {
  const rateMap = {
    'Line 1': product.l1_rate || 0,
    'Line 2': product.l2_rate || 0,
    'Line 3': product.l3_rate || 0,
    'Line 4': product.l4_rate || 0,
    'Line 5': product.l5_rate || 0,
    'Line 6': product.l6_rate || 0,
    'Line 7': product.l7_rate || 0,
  };
  
  // Feedmill lines with rate > 0
  const feedmillAvailable = feedmillLines
    .filter(line => rateMap[line] > 0)
    .map(line => ({ line, rate: rateMap[line] }))
    .sort((a, b) => b.rate - a.rate);
  
  // Other feedmill lines with rate > 0
  const otherLines = Object.entries(rateMap)
    .filter(([line, rate]) => rate > 0 && !feedmillLines.includes(line))
    .map(([line, rate]) => ({ line, rate: parseFloat(rate) }))
    .sort((a, b) => b.rate - a.rate);
  
  const bestLine = feedmillAvailable[0] || null;
  const alsoAvailable = feedmillAvailable.slice(1);
  
  return { bestLine, alsoAvailable, otherLines };
}
```

| Line 6 | Feedmill 3 | Batch FM3 |
| Line 7 | Powermix | Batch PMX |

### BEHAVIOR WHEN LINE CHANGES

When the user changes the Line selection, ALL dependent fields recalculate:

1. **Batch Size** → updates from KB (FM1, FM2, FM3, or PMX)
2. **Batches** → recalculates: Volume ÷ new Batch Size
3. **HA Info** → updates to match new Batches
4. **Production Time** → recalculates: new Batches × new Run Rate
5. **Bags** → recalculates based on new batches
6. **Divisibility warning** → re-evaluates with new Batch Size

**Example:**
```
Product: Lakas Gatas Pellet 50kg
Volume: 80 MT

User selects Line 1 (Feedmill 1):
  Batch Size = 4.00 (Batch FM1)
  Batches = 80 ÷ 4 = 20
  Run Rate = 20.13 min/batch (L1 Rate)
  Production Time = 20 × 20.13 = 402.6 min = 6.71 hrs

User changes to Line 6 (Feedmill 3):
  Batch Size = 2.00 (Batch FM3)
  Batches = 80 ÷ 2 = 40
  Run Rate = 7.16 min/batch (L6 Rate)
  Production Time = 40 × 7.16 = 286.4 min = 4.77 hrs
```
