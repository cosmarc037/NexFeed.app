## 3. FIX: FULFILLMENT / GAP CALCULATION

The fulfillment percentages are incorrect — showing values like 1707%, 1061%, 295% which don't make sense.

### THE CORRECT CALCULATION

```javascript
function calculateGapPercentage(product) {
  const inventory = product.inventory;
  const dueForLoading = product.due_for_loading;
  
  if (!inventory || inventory === 0) {
    return { buffer: 0, status: 'no_inventory' };
  }
  
  // Buffer = how much of inventory remains after loading
  // Buffer% = (Inventory - Due for Loading) / Inventory × 100
  const buffer = ((inventory - dueForLoading) / inventory) * 100;
  
  return {
    buffer: Math.round(buffer * 10) / 10, // 1 decimal
    status: buffer > 20 ? 'sufficient_buffer' : 
            buffer > 0 ? 'low_buffer' : 'deficit'
  };
}
```

Buffer = (376.4 - 127.8) / 376.4 × 100 = 66.1%
→ 66% buffer remaining

Buffer = (217.6 - 20.5) / 217.6 × 100 = 90.6%
→ 91% buffer remaining

```javascript
function renderAIText(rawText) {
  let html = rawText;
  
  // Convert **bold** to <strong>
  html = html.replace(/\*\*(.*?)\*\*/g, '<strong>$1</strong>');
  
  // Convert *italic* to <em>
  html = html.replace(/\*(.*?)\*/g, '<em>$1</em>');
  
  // Remove any leftover link syntax [text](code)
  html = html.replace(/\[(.*?)\]\([\d]*\)/g, '<strong>$1</strong>');
  
  // Remove any raw material codes in parentheses
  html = html.replace(/\(\d{10,}\)/g, '');
  
  // Ensure line breaks are rendered
  html = html.replace(/\n\n/g, '</p><p>');
  html = html.replace(/\n/g, '<br>');
  
  // Convert bullet points to proper list items
  html = html.replace(/• /g, '<br>• ');
  
  // Clean up any double breaks
  html = html.replace(/<br><br>/g, '<br>');
  
  return html;
}
```
