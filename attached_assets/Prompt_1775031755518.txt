## DONE ORDERS — TOP 3 RECENT, FIFO ROTATION

Only the 3 most recent Done orders are shown at the top of the order table per line. When a 4th order is marked as Done, the oldest Done order is removed from the table and moved to Order History.

---

### THE RULE

```
Maximum 3 Done orders visible at the top of each line at any time.
Sorted by completion timestamp: oldest → middle → newest (top to bottom).
When a new order is marked Done, the oldest Done is pushed out to Order History.
```

### ENFORCE TOP 3 RULE

```javascript
function enforceTop3DoneRule(allOrders, currentLine) {
  const MAX_VISIBLE_DONE = 3;
  
  // Get all Done orders on this line
  const doneOrders = allOrders.filter(
    o => o.status === 'Done' && o.line === currentLine
  );
  
  if (doneOrders.length <= MAX_VISIBLE_DONE) {
    // 3 or fewer — no action needed
    return;
  }
  
  // Sort by completion timestamp — oldest first
  doneOrders.sort((a, b) => {
    const timeA = a._doneTimestamp ? new Date(a._doneTimestamp).getTime() : 0;
    const timeB = b._doneTimestamp ? new Date(b._doneTimestamp).getTime() : 0;
    return timeA - timeB;
  });
  
  // Remove oldest Done orders beyond the limit
  const ordersToRemove = doneOrders.slice(0, doneOrders.length - MAX_VISIBLE_DONE);
  
  ordersToRemove.forEach(order => {
    // Move to Order History
    moveToOrderHistory(order);
    
    // Remove from order table
    const index = allOrders.findIndex(o => o.id === order.id);
    if (index !== -1) {
      allOrders.splice(index, 1);
    }
  });
  
  // Renumber priorities
  allOrders.forEach((o, i) => {
    o.prio = i + 1;
  });
}
```

### DONE ORDER SORTING AT TOP

The 3 visible Done orders are always sorted by timestamp — oldest first, newest last:

```javascript
function sortDoneOrdersAtTop(allOrders, currentLine) {
  const doneOrders = allOrders.filter(
    o => o.status === 'Done' && o.line === currentLine
  );
  const nonDoneOrders = allOrders.filter(
    o => o.status !== 'Done' || o.line !== currentLine
  );
  
  // Sort Done by timestamp — oldest first
  doneOrders.sort((a, b) => {
    const timeA = a._doneTimestamp ? new Date(a._doneTimestamp).getTime() : 0;
    const timeB = b._doneTimestamp ? new Date(b._doneTimestamp).getTime() : 0;
    return timeA - timeB;
  });
  
  // Done at top, then non-done in their existing order
  const result = [...doneOrders, ...nonDoneOrders];
  result.forEach((o, i) => {
    o.prio = i + 1;
  });
  
  return result;
}
