## NOTIFICATION TIMERS — DYNAMIC DURATION BASED ON TYPE AND LENGTH

Notifications in the upper-right should have different auto-dismiss durations based on their type and text length.

### DURATION RULES

#### BASE DURATION BY TYPE

| Type | Base Duration | Reasoning |
|---|---|---|
| Success | 3 seconds | Short, confirmatory — user just needs a glance |
| Info | 4 seconds | Slightly longer — may contain useful context |
| Warning | 6 seconds | Important — user should read and understand |
| Error | 8 seconds | Critical — user needs time to read and may need to act |

#### DYNAMIC DURATION — BASED ON TEXT LENGTH

Add extra time for longer messages. For every 50 characters beyond 80 characters, add 1 second:

```javascript
function getNotificationDuration(type, message) {
  // Base durations (milliseconds)
  const baseDurations = {
    success: 3000,
    info: 4000,
    warning: 6000,
    error: 8000
  };
  
  const base = baseDurations[type] || 4000;
  
  // Add time for longer messages
  const messageLength = (message || '').length;
  let extraTime = 0;
  
  if (messageLength > 80) {
    const extraChars = messageLength - 80;
    extraTime = Math.ceil(extraChars / 50) * 1000; // 1 second per 50 chars
  }
  
  // Cap at maximum duration
  const maxDurations = {
    success: 5000,
    info: 7000,
    warning: 10000,
    error: 15000
  };
  
  const maxDuration = maxDurations[type] || 10000;
  
  return Math.min(base + extraTime, maxDuration);
}
```

### NOTIFICATION COMPONENT

```jsx
function Notification({ id, type, title, message, onDismiss }) {
  const duration = getNotificationDuration(type, message);
  const [isExiting, setIsExiting] = useState(false);
  const timerRef = useRef(null);
  const progressRef = useRef(null);
  
  useEffect(() => {
    // Start auto-dismiss timer
    timerRef.current = setTimeout(() => {
      handleDismiss();
    }, duration);
    
    return () => clearTimeout(timerRef.current);
  }, [duration]);
  
  function handleDismiss() {
    setIsExiting(true);
    setTimeout(() => {
      onDismiss(id);
    }, 300); // Exit animation duration
  }
  
  // Pause timer on hover
  function handleMouseEnter() {
    clearTimeout(timerRef.current);
  }
  
  // Resume timer on mouse leave
  function handleMouseLeave() {
    timerRef.current = setTimeout(() => {
      handleDismiss();
    }, 2000); // Give 2 more seconds after hover
  }
  
  const icons = {
    success: '✓',
    info: 'ℹ',
    warning: '⚠',
    error: '✕'
  };
  
  const colors = {
    success: { bg: '#f0fdf4', border: '#bbf7d0', icon: '#16a34a', progress: '#16a34a' },
    info: { bg: '#eff6ff', border: '#bfdbfe', icon: '#2563eb', progress: '#2563eb' },
    warning: { bg: '#fffbeb', border: '#fde68a', icon: '#ca8a04', progress: '#ca8a04' },
    error: { bg: '#fef2f2', border: '#fecaca', icon: '#dc2626', progress: '#dc2626' }
  };
  
  const style = colors[type] || colors.info;
  
  return (
    <div
      className={`notification ${type} ${isExiting ? 'exiting' : ''}`}
      style={{
        background: style.bg,
        border: `1px solid ${style.border}`,
      }}
      onMouseEnter={handleMouseEnter}
      onMouseLeave={handleMouseLeave}
    >
      <div className="notification-content">
        <span className="notification-icon" style={{ color: style.icon }}>
          {icons[type]}
        </span>
        
        <div className="notification-text">
          {title && (
            <div className="notification-title">{title}</div>
          )}
          <div className="notification-message">{message}</div>
        </div>
        
        <button 
          className="notification-close"
          onClick={handleDismiss}
        >
          ✕
        </button>
      </div>
      
      {/* Progress bar — shows remaining time */}
      <div 
        className="notification-progress"
        style={{
          background: style.progress,
          animationDuration: `${duration}ms`
        }}
      />
    </div>
  );
}
```
