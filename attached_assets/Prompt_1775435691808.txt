## FIX: SHOW WARNING WHENEVER A PLANNED ORDER'S POSITION CHANGES

A warning confirmation dialog must appear whenever a Planned order's priority position changes — regardless of HOW it changes. Whether the Planned order is:
- Dragged directly by the user
- Pushed DOWN because another order was inserted above it
- Pushed UP because another order was inserted below and displaces it

If the Planned order ends up at a different priority than where it was, show the warning.

---

### THE PROBLEM

Currently:
- ✅ Dragging a non-planned order directly to the Planned order's position shows a warning
- ❌ Dragging a non-planned order ABOVE the Planned order (e.g., to Prio 1) pushes Planned down but NO warning
- ❌ Dragging a non-planned order BELOW the Planned order, causing Planned to shift up, shows NO warning
- ❌ Dragging the Planned order itself to another position shows NO warning (or inconsistent)

### THE FIX

After ANY drag-and-drop operation completes, check if ANY Planned order's position has changed. If yes, show the warning.

```javascript
function handleDragEnd(draggedOrderId, fromIndex, toIndex) {
  if (fromIndex === toIndex) return; // No movement
  
  // Step 1: Record all Planned orders' current positions BEFORE the move
  const plannedOrdersBefore = {};
  allOrders.forEach((order, index) => {
    if (order.status === 'Planned') {
      plannedOrdersBefore[order.id] = {
        order: order,
        prio: index + 1,
        name: order.item_description
      };
    }
  });
  
  // Step 2: Perform the reorder in a temporary array
  const reorderedOrders = [...allOrders];
  const [movedOrder] = reorderedOrders.splice(fromIndex, 1);
  reorderedOrders.splice(toIndex, 0, movedOrder);
  
  // Step 3: Check all Planned orders' positions AFTER the move
  const affectedPlannedOrders = [];
  reorderedOrders.forEach((order, index) => {
    if (order.status === 'Planned') {
      const newPrio = index + 1;
      const before = plannedOrdersBefore[order.id];
      
      if (before && before.prio !== newPrio) {
        affectedPlannedOrders.push({
          order: order,
          name: order.item_description,
          fromPrio: before.prio,
          toPrio: newPrio,
          direction: newPrio > before.prio ? 'down' : 'up'
        });
      }
    }
  });
  
  // Step 4: If any Planned order is affected, show warning
  if (affectedPlannedOrders.length > 0) {
    showPlannedOrderWarning(affectedPlannedOrders, () => {
      // User confirmed — apply the reorder
      applyReorder(reorderedOrders);
    }, () => {
      // User cancelled — revert, do nothing
    });
  } else {
    // No Planned orders affected — apply immediately
    applyReorder(reorderedOrders);
  }
}

function applyReorder(reorderedOrders) {
  // Update priorities
  reorderedOrders.forEach((order, index) => {
    order.prio = index + 1;
    
    // Update Planned order's locked position
    if (order.status === 'Planned') {
      order._plannedPrio = index + 1;
    }
  });
  
  setAllOrders(reorderedOrders);
}
```
