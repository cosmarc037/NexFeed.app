## FIX: BACKGROUND CELL HIGHLIGHT — ONLY YELLOW FOR OUTSIDE FEEDMILL DIVERSION

The row background is still showing yellow for ALL divertible orders. It should ONLY be yellow if the order can be diverted to lines OUTSIDE the feedmill. Orders that can only go to the partner line within the same feedmill should have NO background change — just the normal white row with the "right-click to divert" note.

---

### THE RULE — SIMPLE AND CLEAR

| Scenario | Background | Note |
|---|---|---|
| Order can divert to lines OUTSIDE feedmill (e.g., Line 4, Line 7) | ✅ YELLOW background | "⚠ Line shutdown — right-click to divert to other feedmill lines" |
| Order can ONLY divert to partner line WITHIN feedmill (e.g., Line 2) | ❌ NO background change (white) | "↩ Line shutdown — right-click to divert" |
| Order cannot divert anywhere | ❌ NO background change (white) | "🔒 Cannot be diverted — awaiting line resume" |

### EXAMPLE

Line 1 is shutdown. Line 2 is still running.

```

Order B: Has run rate ONLY on Line 2 (partner) → NO highlight, white background + note

### FIX THE ROW CLASS LOGIC

The issue is in the component that assigns the CSS class. Make absolutely sure ONLY `_canDivertOutsideFeedmill === true` triggers the yellow class:

```jsx
function OrderRow({ order, lineShutdowns }) {
  const isLineShutdown = lineShutdowns[order.line]?.isShutdown === true;
  
  // Determine row state
  let showYellowBackground = false;
  let showDivertNote = false;
  let showStuckNote = false;
  let divertNoteText = '';
  let divertNoteClass = '';
  
  if (isLineShutdown && order.status !== 'Done' && order.status !== 'Cancel PO') {
    
    if (order._canDivertOutsideFeedmill === true) {
      // CAN go to lines outside feedmill → YELLOW BACKGROUND
      showYellowBackground = true;
      showDivertNote = true;
      divertNoteText = '⚠ Line shutdown — right-click to divert to other feedmill lines';
      divertNoteClass = 'divert-hint-outside';
      
    } else if (order._canDivertWithinFeedmill === true) {
      // Can ONLY go to partner line → NO BACKGROUND CHANGE
      showYellowBackground = false; // EXPLICITLY false
      showDivertNote = true;
      divertNoteText = '↩ Line shutdown — right-click to divert';
      divertNoteClass = 'divert-hint-within';
      
    } else {
      // Cannot divert anywhere
      showYellowBackground = false;
      showDivertNote = false;
      showStuckNote = true;
    }
  }
  
  // Build row class — ONLY add yellow class if showYellowBackground is true
  let rowClass = 'order-row';
  if (showYellowBackground) {
    rowClass += ' order-row-divertible-outside';
  }
  // DO NOT add any highlight class for white-background orders
  
  const canRightClick = showDivertNote; // Both yellow and white-with-note can right-click
  
  return (
    <tr 
      className={rowClass}
      onContextMenu={(e) => {
        if (canRightClick) {
          e.preventDefault();
          showDivertContextMenu(e, order);
        }
      }}
    >
      {/* ... other cells ... */}
      <td className="item-description-cell">
        <div className="item-desc-name">{order.item_description}</div>
        <div className="item-desc-meta">
          <span className="item-desc-category">{order.category}</span>
          {order.color && (
            <>
              <span className="item-desc-separator">|</span>
              <span className="item-desc-color">{order.color}</span>
            </>
          )}
          {order.diameter && (
            <>
              <span className="item-desc-separator">|</span>
              <span className="item-desc-diameter">{order.diameter}mm</span>
            </>
          )}
        </div>
        
        {/* Divert note — for both yellow and white rows */}
        {showDivertNote && (
          <div className={`item-desc-divert-hint ${divertNoteClass}`}>
            {divertNoteText}
          </div>
        )}
        
        {/* Stuck note — cannot divert */}
        {showStuckNote && (
          <div className="item-desc-divert-hint divert-hint-stuck">
            🔒 Cannot be diverted — awaiting line resume
          </div>
        )}
      </td>
    </tr>
  );
}
