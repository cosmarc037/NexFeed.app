## 3. FIX LAST ORDER CHANGEOVER — NO CHANGEOVER FOR LAST ORDER

The changeover should be 0 for the last order on a line since there's no following order to compare against.

```javascript
// In cascadeSchedule or calculateChangeovers:

function calculateChangeovers(orders, changeoverRules) {
  for (let i = 0; i < orders.length; i++) {
    const currentOrder = orders[i];
    
    // LAST ORDER — no changeover
    if (i === orders.length - 1) {
      currentOrder.changeover = 0;
      currentOrder._changeoverTotal = 0;
      currentOrder._changeoverDetails = null;
      continue;
    }
    
    // Not last — calculate based on the NEXT order
    const nextOrder = orders[i + 1];
    const changeover = calculateChangeoverBetween(currentOrder, nextOrder, changeoverRules);
    currentOrder.changeover = changeover;
    currentOrder._changeoverTotal = changeover;
  }
}
```
