## 2. CREATE A HELPER TO SUM TOTAL MT PER LINE

Use a shared helper to compute total MT for a given line.

```javascript
function getOrderVolumeMT(order) {
  const candidates = [
    order.volume_override,
    order.volume,
    order.total_volume_mt,
    order.volume_mt,
  ];

  for (const val of candidates) {
    const parsed = parseFloat(val);
    if (!Number.isNaN(parsed)) return parsed;
  }

  return 0;
}

function calculateLineTotalMT(orders) {
  return Number(
    (orders || []).reduce((sum, order) => {
      return sum + getOrderVolumeMT(order);
    }, 0).toFixed(2)
  );
}
```

This helper must be used consistently for queue-time calculations.

## 4. CALCULATE SOURCE AND DESTINATION LOADS FROM THE ACTUAL PRE-MOVE STATE

When evaluating a move from source line to destination line:

### Before move
- source line total MT = sum of orders currently on source line
- destination line total MT = sum of orders currently on destination line

### After move
- source line total MT after move = source total MT before move − moved order MT
- destination line total MT after move = destination total MT before move + moved order MT

This must be explicit.

### Example:
```javascript
const movedMT = getOrderVolumeMT(orderToMove);

const sourceBeforeMT = calculateLineTotalMT(sourceLineOrders);
const destinationBeforeMT = calculateLineTotalMT(destinationLineOrders);

const sourceAfterMT = Number((sourceBeforeMT - movedMT).toFixed(2));
const destinationAfterMT = Number((destinationBeforeMT + movedMT).toFixed(2));
```

Do not accidentally compute sourceBeforeMT from a list where the moved order has already been removed.

## 6. CREATE A QUEUE TIME HELPER

function calculateQueueTimeHours(totalMT, runRate) {
  const mt = parseFloat(totalMT) || 0;
  const rr = parseFloat(runRate) || 0;

}
