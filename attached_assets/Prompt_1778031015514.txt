## 1. LOOK UP COST AND MARGIN USING MATERIAL CODE

For every order row:
- read the order’s material code
- match it against Master Data
- pull:
  - Cost
  - Margin

Use the material code as the lookup key.

### Matching rules
Normalize the material code before matching:
- convert to string
- trim whitespace
- compare consistently

### Suggested helper:
```javascript
function normalizeMaterialCode(value) {
  if (value == null) return '';
  return String(value).trim();
}

function getPricingFromMasterData(order, masterData) {
  const orderCode = normalizeMaterialCode(order.material_code_fg || order.material_code);

  if (!orderCode) {
    return {
      cost: null,
      margin: null,
      pricingFound: false,
    };
  }

  const match = (masterData || []).find(md => {
    const codesToCheck = [
      md.material_code_fg,
      md.material_code,
      md.MaterialCode,
      md['Material Code'],
      md['Material Code (FG)'],
    ].map(normalizeMaterialCode);

    return codesToCheck.includes(orderCode);
  });

  if (!match) {
    return {
      cost: null,
      margin: null,
      pricingFound: false,
    };
  }

  const cost = match.Cost ?? match.cost ?? match['Cost'] ?? null;
  const margin = match.Margin ?? match.margin ?? match['Margin'] ?? null;

  return {
    cost,
    margin,
    pricingFound: true,
  };
}
```
