## 2. CALCULATE THE HOURS BREAKDOWN PER LINE

Create a shared helper that returns:
- production only
- changeover only
- total including changeover

### Suggested helper:
```javascript
function calculateLineHoursBreakdown(orders) {
  const productionHours = Number(
    (orders || []).reduce((sum, order) => {
      return sum + getOrderProductionHours(order);
    }, 0).toFixed(2)
  );

  const changeoverHours = Number(
    (orders || []).reduce((sum, order) => {
      return sum + getOrderChangeoverHours(order);
    }, 0).toFixed(2)
  );

  const totalHours = Number((productionHours + changeoverHours).toFixed(2));

  return {
    productionHours,
    changeoverHours,
    totalHours,
  };
}
```
