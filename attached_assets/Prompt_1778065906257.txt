- do not persist error state to the next run
- do not build fallback “AI unavailable” cards unless the current run truly failed for a non-cancel reason

### Example:

```javascript
    err?.name === 'AbortError' ||
    err?.message === 'AUTO_SEQUENCE_CANCELLED'

    return { cancelled: true };
  }

  // real failure only
  throw err;
}
```

Then ensure cancelled runs do not write AI failure state into the UI.
