## 1. DETERMINE THE PRIMARY REASON FOR LINE PLACEMENT BEFORE BUILDING THE TEXT

Before generating the move insight, determine whether the move was driven by:
- line eligibility from Master Data
- lower queue time
- lower changeover burden
- balancing logic
- other sequencing constraints

### Example helper:
```javascript
function getPlacementReasonContext({
  order,
  sourceLine,
  targetLine,
  eligibleLines,
}) {
  const normalizedEligible = (eligibleLines || []).map(normalizeLineKey);
  const src = normalizeLineKey(sourceLine);
  const tgt = normalizeLineKey(targetLine);

  const sourceEligible = normalizedEligible.includes(src);
  const targetEligible = normalizedEligible.includes(tgt);
  const onlyTargetEligible = targetEligible && normalizedEligible.length === 1;

  return {
    sourceEligible,
    targetEligible,
    onlyTargetEligible,
  };
}
```
