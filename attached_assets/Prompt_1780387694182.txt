## 1. MAKE THE `REASON` COLUMN AI-GENERATED

Please change the **Reason** column in the **AI vs Final Sequence — Adjustment Details** table so that it is generated by AI rather than relying only on generic/static template text.

### Goal
Provide more meaningful row-specific explanations for:
- why the order is placed where it is
- why it was or was not moved
- why the final rank differs from the AI rank, if applicable

### Expected result
Each row should show a more natural, contextual explanation instead of only generic repeated text.

---

## 2. MAKE THE AI EXPLAIN THE ACTUAL PLACEMENT OF THE ORDER

The AI-generated reason should explain the actual placement of the order in the final sequence.

### Examples of what the reason should cover
Depending on the row, explain things like:
- why the order stayed where it is
- why it moved earlier or later
- why it was pinned in place
- why it could not follow the raw AI rank
- what rule or sequencing condition influenced the final position
- how neighboring orders / compatibility / deadlines / line constraints affected placement

### Goal
Help the user understand the sequencing decision in practical plain language.

### Expected result
The reason feels specific to that row, not like a generic label.

---

## 3. KEEP THE CURRENT TEMPLATE/GNERIC REASON AS A FALLBACK ONLY

Please keep the existing template-based/generic reason logic only as a fallback.

### Use fallback when:
- AI reason generation fails
- AI output is unavailable
- request times out
- response is malformed
- or another issue prevents valid AI text from being returned

### Expected result
The app remains reliable even if AI-generated reasoning is unavailable, but uses richer AI explanations whenever possible.

---

## 4. KEEP `CONSTRAINT NOTE` AS THE STRUCTURED / DEBUGGABLE RULE LABEL

Please preserve the `Constraint Note` column as the structured machine-readable rule/category indicator.

### Example values

- etc.

### Goal
Keep a clean separation between:
