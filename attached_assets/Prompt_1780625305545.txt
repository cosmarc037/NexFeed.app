Please create a derived **Velocity** classification based on how stable or erratic the daily demand is over the next 10 days.

### Column name
- **Velocity**

### Allowed values
- **Stable**
- **Less Stable**
- **Erratic**

### Suggested basis
Use the daily demand values already present in the next 10-day forecast columns.

### Recommended method
Calculate demand variability using the daily forecast values, for example by using:
- mean
- standard deviation
- coefficient of variation (CV)

Where:

- `CV = standard deviation / mean`

### Suggested classification
- **Stable** → low variability
- **Less Stable** → moderate variability
- **Erratic** → high variability

### Example threshold idea
- Stable → `CV <= 0.5`
- Less Stable → `0.5 < CV <= 1.0`
- Erratic → `CV > 1.0`

You may adjust thresholds as needed if a slightly different calibration works better for the available data.

### Expected result

- Stable
- Less Stable
- Erratic

based on how variable its upcoming daily demand pattern is.
