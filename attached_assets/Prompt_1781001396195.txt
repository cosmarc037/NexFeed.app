## 1. INCLUDE `ADD ORDER` CREATION EVENTS IN ORDER HISTORY

Please make sure that when a user creates a new order using the **Add Order** feature, that action is recorded in the order history.

### Important
This is specifically about:
- manually adding a new order through the **Add Order** feature

This is **not** a general request for every possible type of order creation/import.

### Goal
Make manually created orders traceable from the moment they are added through the Add Order flow.

### Expected result
When a user adds a new order, the history shows a clear event such as:
- `Order Created`
- or `Added via Add Order`
